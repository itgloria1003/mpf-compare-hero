CMSC5702 – Assignment 1: Distributed Systems Programming

MPF Compare Hero

MPF Compare Hero is an application that pull data from other websites and convert them into structured data. The application has the following features.

List the MPF fund type categories

List registered MPF trustees and MPF schemes

Compare the performance for the MPF schemes


Learn more on the approach

The approach

There are various ways to extract information from the web, including RMI ,REStful API , Webimplementation.

In this assignment we used web scraping approach to pull the data from the websites, including MPFA , Securities & Futures Commissions website, and then convert into some structured data and stored in the server in-memory cache.
